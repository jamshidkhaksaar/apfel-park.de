# Project Context: Apfel (Next.js)

## Overview
This is a fresh Next.js application initialized on January 11, 2026. The previous WordPress project has been removed.

## Tech Stack
-   **Framework:** Next.js (Latest)
-   **Language:** TypeScript
-   **Styling:** Tailwind CSS
-   **Linting:** ESLint
-   **Deployment/Containerization:** Docker (Dockerfile and docker-compose.yml provided)

## Project Structure
-   `src/`: Source code directory (App Router)
-   `public/`: Static assets
-   `next.config.ts`: Next.js configuration
-   `Dockerfile`: Docker build instructions for production
-   `docker-compose.yml`: Docker orchestration

## Development
-   **Run locally:** `npm run dev`
-   **Build for production:** `npm run build`
-   **Run in Docker:** `docker-compose up --build`
